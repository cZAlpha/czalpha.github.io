Personal Projects

Plasmainterferometry

The Plasmainterferometry project is a Python program designed to measure the density of plasma and gas using laser fringe patterns. Users take images of these patterns, trace the fringe lines, and convert the images into a special format. The program processes these images to analyze the density of the plasma or gas. By running the main program file and selecting the desired analysis options, users can generate various graphs to visualize the results. The process involves image preparation, tracing, and saving the lines in a specific format before using the program to complete the analysis.

Food Ordering App

This programming project includes an IOS application made using React Native that you are able to use through XCode's simulator. It allows for authorization for admin logins, general public use, etc. It allows you to upload images from your machine through the application in order to create and/or edit products, their names and prices. Furthermore, users are able to order products and those products will appear in their cart.
