Lumetric

Private

Lumetric is a web-based assessment tool designed to help DSU professors streamline accreditation processes and reduce time spent on administrative tasks. It enables educators to input courses and sections, upload evaluation tools used in their courses, and automatically analyze student performance against course and program learning objectives (CLOs and PLOs). This project's repository is private for legal reasons, please contact me if you have any interest or questions.

Farmer AI

FarmerAI is a Python-based chatbot application designed to answer agriculture-related questions for farmers, gardeners, and agricultural enthusiasts. This interactive tool leverages advanced language models to provide reliable information on topics like crop management, pest control, and irrigation techniques.

Plasmainterferometry

The Plasmainterferometry project is a Python program designed to measure the density of plasma and gas using laser fringe patterns. Users take images of these patterns, trace the fringe lines, and convert the images for analysis to visualize plasma or gas density results.

PyFile Converter

The Pyfile Converter application is a Python based application mainly made for Mac OS that is able to convert text and image files, with video file conversion coming soon. It is open source and welcomes contributions from other developers.

Food Ordering App

This programming project includes an IOS application made using React Native that you are able to use through XCode's simulator. It allows for authorization for admin logins, general public use, etc. It allows you to upload images from your machine through the application in order to create and/or edit products, their names and prices. Furthermore, users are able to order products and those products will appear in their cart.
